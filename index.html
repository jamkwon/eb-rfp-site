1.2 KPI Hierarchy

Our proposed metric priority hierarchy is:

  1. Cost-per-Hire (CPH). This is the true measure of success: how efficiently media investment turns into actual hires.
  2. Screening-Qualified Applicant rate (SQA). This measures how many applicants meet baseline qualifications.
  3. Cost-per-Application (CPA). CPA tells us how efficiently we are generating applicants, but it must be evaluated alongside quality (SQA), not in isolation.
  4. Application completion rate. This indicates whether candidates who show intent can complete the application process without friction.
  5. Reach, frequency, and awareness. These metrics help us understand whether employer messaging is reaching enough people.

1.3 Guardrails to Prevent Budget Fragmentation

We will start with the smallest set of channels that reliably produce a qualified applicant flow, then expand only when it compounds results. Protect Search + retargeting first. Flight awareness; don't "always-on" it out of habit.
